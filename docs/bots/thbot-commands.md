# 叹号Bot 指令列表

叹号Bot (以花开头的bot，如花沐) 指令列表，提供了群管理、娱乐等多种功能。

---

::: warning

部分指令需要权限才能使用

:::

::: tip

使用 `指令名 -h` 查看特定指令的语法和使用示例。

:::

---

::: details 点击此处查看本页目录
[[toc]]
:::

## 📋 用户管理
| 指令 | 说明 |
|------|------|
| `admin` | 设置管理员 |
| `unadmin` | 取消管理员 |
| `ban` | 禁言用户 |
| `unban` | 解除用户禁言 |
| `ban-all` | 全体禁言 |
| `unban-all` | 解除全体禁言 |
| `unban-allppl` | 解除所有人禁言 |
| `unban-batch` | 批量解除禁言 |
| `unban-random` | 随机解除若干人禁言 |
| `ban-list` | 查询当前禁言名单 |
| `banme` | 随机禁言自己 |
| `kick` | 踢出用户 |
| `warn` | 警告用户 |
| `stop` | 短期禁言 |
| `user` | 用户管理 |
| `nickname` | 设置用户昵称 |
| `callme` | 修改自己的称呼 |
| `getauth` | 获取指定成员状态喵 |
| `gauth` | 管理用户角色 |
