# 叹号Bot 指令列表

叹号Bot (以花开头的bot，如花沐) 指令列表，提供了群管理、娱乐等多种功能。

---

::: warning

部分指令需要权限才能使用

:::

::: tip

使用 `指令名 -h` 查看特定指令的语法和使用示例。

:::

---

::: details 点击此处查看本页目录
[[toc]]
:::

## 群管理指令

### 管理员管理

| 指令 | 说明 |
|------|------|
| `admin` | 设置管理员 |
| `unadmin` | 取消管理员 |
| `gauth` | 管理用户角色 |
| `getauth` | 获取指定成员状态 |

### 禁言管理

| 指令 | 说明 |
|------|------|
| `ban` | 禁言用户 |
| `unban` | 解除用户禁言 |
| `ban-all` | 全体禁言 |
| `unban-all` | 解除全体禁言 |
| `ban-list` | 查询当前禁言名单 |
| `banme` | 随机禁言自己 |
| `unban-allppl` | 解除所有人禁言 |
| `unban-batch` | 批量解除禁言 |
| `unban-random` | 随机解除若干人禁言 |
| `stop` | 短期禁言 |
| `forbidden` | 禁言关键词管理 |

### 用户管理

| 指令 | 说明 |
|------|------|
| `kick` | 踢出用户 |
| `warn` | 警告用户 |
| `nickname` | 设置用户昵称 |
| `title` | 群头衔管理 |
| `user` | 用户管理 |
| `verify` | 入群验证关键词管理 |
| `inspect` | 查看用户、频道或消息的详细信息 |

### 其他管理

| 指令 | 说明 |
|------|------|
| `channel` | 频道管理 |
| `grouphelper` | 群管理帮助 |
| `welcome` | 入群欢迎语管理 |
| `goodbye` | 退群欢送语管理 |
| `quit-group` | 退出指定群聊 |
