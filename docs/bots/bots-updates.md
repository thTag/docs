## Lar（OneBot）

> 含 AI 对话功能的Bot Lar

### 新增丨New Features

- 新功能: 生成奖状 (`奖状`) 给优秀群友颁发一张奖状吧！

- 新功能：中国象棋 (`cchess`) 和 群友/AI 来场中国象棋对战吧

- 新特性：支持分析视频 (`zssm`) 啦！引用视频发送 zssm 即可分析（总结）视频内容

- 新用法：已经支持解析 `微博` `小红书` `快手` `小黑盒` `A站` `NGA` `Tiktok` `Twitter` 链接啦！

### 优化丨Improvements

- 更换 表情包制作 (`meme`) 引擎，提升表情包制作速度

### 修复丨Bug Fixes

- 修复 群分析（`群分析`）功能无法返回精美的图片的 bug

- 移除 鹿 签到 (`鹿管签到`) 功能
